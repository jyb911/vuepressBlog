**二叉排序树**（**二叉搜索树**）：

- 如果它的左子树不为空，那么左子树上的所有结点的值均小于它的根结点的值
- 如果它的右子树不为空，那么右子树上的左右结点的值均大于它的根结点的值
- 根结点的左子树和右子树又是二叉排序树。
- 中序遍历为升序
